# 为什么增加博查（Bocha AI）

## 背景

根据 [36氪报道](https://36kr.com/p/3254160451907584)，博查（Bocha AI）是一个明确使用了 **Bocha AI Web Search API**（https://open.bochaai.com/）的搜索引擎，并且可能使用了 Bing API 作为底层技术支持。

## 核心目的

增加博查（Bocha AI）作为对比分析工具，主要目的如下：

### 1. 多引擎结果对比

通过整合博查与其他大模型搜索引擎（如 Bing、Google 等），可以：

- **横向对比**：比较不同搜索引擎对同一查询的返回结果
- **结果验证**：交叉验证搜索结果的一致性和准确性
- **质量评估**：识别不同引擎的优势和局限性

### 2. 相似度检测与 GEO 处理

当检测到以下情况时：

- **博查结果匹配**：搜索结果明确来自博查或者与博查结果高度相似
- **高相似度判断**：通过相似度算法判断结果与博查的匹配度达到阈值

则可以直接对博查进行 **GEO（地理位置）处理**，优化地理营销策略。

### 3. 优势分析

- **API 可用性**：博查提供了公开的 Web Search API，便于集成
- **结果独特性**：博查可能使用 Bing API 等底层技术，提供差异化的搜索结果
- **地理营销优化**：针对博查搜索结果进行 GEO 处理，可以更精准地进行地理定向营销

## 技术实现

- **API 端点**：https://open.bochaai.com/
- **底层技术**：可能使用 Bing API
- **集成方式**：通过对比分析模块与其他搜索引擎结果进行横向比较

## 预期效果

通过增加博查搜索引擎，可以实现：

1. 更全面的搜索结果覆盖
2. 更准确的相似度判断
3. 更精准的地理营销策略制定
