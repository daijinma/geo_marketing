# 语义足迹扩展文档

本目录包含语义足迹扩展相关的文档、模板和示例。

## 目录结构

- `README.md` - 本文件
- `话题映射模板.md` - 话题映射工作模板
- `内容规划矩阵模板.csv` - 内容规划矩阵模板
- `Prompt映射模板.md` - Prompt 映射模板
- `装修行业话题地图.md` - 装修行业示例

## 使用说明

### 1. 话题映射

1. 使用 `话题映射模板.md` 作为起点
2. 填写核心关键词
3. 使用 LLM 生成邻近话题
4. 输出到相应行业的话题地图文件

### 2. 内容规划

1. 基于话题地图，填写 `内容规划矩阵模板.csv`
2. 规划每个话题的内容类型、优先级、状态
3. 输出内容规划矩阵

### 3. Prompt 映射

1. 收集用户真实提问
2. 使用 `Prompt映射模板.md` 记录问题模式
3. 针对性地优化页面段落

## 工作流程

```
核心关键词
  ↓
话题邻近性映射 → 生成话题地图
  ↓
内容规划矩阵 → 规划内容生产
  ↓
Prompt 映射 → 优化页面段落
  ↓
内容生产与优化
```
