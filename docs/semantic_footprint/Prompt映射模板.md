## 三、页面段落优化映射

### 3.1 优化策略

针对每个高频问题，确保页面包含：

1. **直接回答段落**：明确回答用户问题
2. **数据支撑**：提供统计数据或事实依据
3. **引用来源**：标注信息来源

### 3.2 优化映射表

| 问题 | 目标页面URL | 优化段落位置 | 优化内容要点 | 状态 |
|-----|------------|-------------|-------------|------|
| [问题] | [页面URL] | [段落位置，如：H2标题下第一段] | [内容要点列表] | [pending/done] |
| ... | ... | ... | ... | ... |

**示例**：
- **问题**：北京装修多少钱？
- **目标页面**：https://example.com/beijing-decorate-price
- **优化段落位置**：H2 "北京装修价格" 下第一段
- **优化内容要点**：
  - 直接回答：根据2024年数据，北京90平米住宅平均装修成本为15-25万元
  - 数据支撑：引用国家统计局或北京市住建委数据
  - 引用来源：标注数据来源链接
- **状态**：pending

## 四、Prompt 模式库

### 4.1 标准 Prompt 模板

#### 如何型 Prompt

```
基于关键词 "{关键词}"，用户可能提出以下问题：
- 如何选择 {关键词}？
- 如何使用 {关键词}？
- 如何避免 {关键词} 的常见问题？

针对这些问题，页面应该包含：
1. 直接回答段落
2. 步骤/流程说明
3. 注意事项
```

#### 为什么型 Prompt

```
基于关键词 "{关键词}"，用户可能提出以下问题：
- 为什么 {关键词} 会出现 {现象}？
- 为什么 {关键词} 价格差异大？

针对这些问题，页面应该包含：
1. 原因分析段落
2. 数据支撑
3. 专家观点引用
```

#### 多少钱型 Prompt

```
基于关键词 "{关键词}"，用户可能提出以下问题：
- {关键词} 多少钱？
- {关键词} 价格是多少？

针对这些问题，页面应该包含：
1. 价格范围直接回答
2. 影响价格的因素
3. 最新统计数据
4. 数据来源标注
```
