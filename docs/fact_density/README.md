# 事实密度提升文档

本目录包含事实密度提升相关的文档、模板和数据收集清单。

## 目录结构

- `README.md` - 本文件
- `数据源收集清单模板.yaml` - 数据源收集清单模板
- `装修行业数据收集清单.yaml` - 装修行业示例
- `引用格式规范.md` - 引用格式规范
- `质量评分标准.md` - 质量评分标准

## 使用说明

### 1. 数据源收集

1. 使用 `数据源收集清单模板.yaml` 作为起点
2. 按照数据源分类填写清单
3. 记录数据来源、采集方法、更新频率

### 2. 引用格式规范

1. 参考 `引用格式规范.md`
2. 确保所有引用符合规范
3. 统一标注数据来源

### 3. 质量评分

1. 参考 `质量评分标准.md`
2. 对收集的数据源进行评分
3. 优先使用高质量数据源

## 工作流程

```
确定数据需求
  ↓
收集数据源清单
  ↓
数据采集与验证
  ↓
数据清洗与标准化
  ↓
存储到数据库
  ↓
质量评分
  ↓
应用到内容增强
```
