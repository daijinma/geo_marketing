# 引用格式规范

## 一、统计数据引用格式

### 1.1 官方统计数据

**格式**：
```markdown
> 根据 [来源名称 发布时间](来源链接) 数据显示，
> [数据项名称] 为 [数据值][数据单位]。
```

**示例**：
```markdown
> 根据 [国家统计局 2024 年度报告](https://www.stats.gov.cn/) 数据显示，
> 2024 年全国住宅装修市场规模为 15000 亿元。

> 根据 [北京市住建委 2024 年第三季度报告](https://zjw.beijing.gov.cn/) 数据显示，
> 北京地区 90 平米住宅平均装修成本为 15-25 万元。
```

### 1.2 行业报告数据

**格式**：
```markdown
> 根据 [来源名称 报告名称 发布时间](来源链接) 报告显示，
> [数据项名称] 为 [数据值][数据单位]。
```

**示例**：
```markdown
> 根据 [艾瑞咨询 2024 年中国装修行业用户调研报告](https://www.iresearch.com.cn/) 报告显示，
> 用户选择装修公司时最关注的因素是价格和服务质量。
```

## 二、引用文献格式

### 2.1 国家标准

**格式**：
```markdown
根据 [标准编号 标准名称](标准链接) 规定，
[相关规定内容]。
```

**示例**：
```markdown
根据 [《住宅装饰装修工程施工规范》(GB 50327-2001)](https://www.mohurd.gov.cn/) 规定，
装修工程必须按照相关标准进行施工和验收。
```

### 2.2 行业标准

**格式**：
```markdown
根据 [标准编号 标准名称](标准链接) 规定，
[相关规定内容]。
```

**示例**：
```markdown
根据 [《住宅室内装饰装修工程质量验收标准》(JGJ/T 304-2013)](https://www.mohurd.gov.cn/) 规定，
装修工程质量验收必须符合相关标准要求。
```

### 2.3 学术论文

**格式**：
```markdown
根据 [作者（年份）](DOI或链接) 的研究表明，
[研究结论]。
```

**示例**：
```markdown
根据 [张三等（2024）](https://doi.org/xxxx) 的研究表明，
住宅装修质量受多种因素影响。
```

### 2.4 专家观点

**格式**：
```markdown
根据 [专家姓名 专家身份](来源链接) 的观点，
[专家观点内容]。
```

**示例**：
```markdown
根据 [李四 资深装修设计师](https://example.com/expert) 的观点，
现代简约风格在北京地区最受欢迎。
```

## 三、独特见解格式

### 3.1 趋势分析

**格式**：
```markdown
> **数据洞察**：基于 [数据来源] 的分析显示，
> [趋势描述]。[数据支撑]
```

**示例**：
```markdown
> **数据洞察**：基于 2020-2024 年北京市住建委历史价格数据的分析显示，
> 北京地区装修成本呈现逐年上涨趋势，年均涨幅约 5-8%。
> 其中，2024 年相比 2023 年上涨 6.5%。
```

### 3.2 数据对比

**格式**：
```markdown
> **数据对比**：对比 [对比维度] 数据显示，
> [对比结论]。[具体数据支撑]
```

**示例**：
```markdown
> **数据对比**：对比北京、上海、广州、深圳四个一线城市的装修成本数据显示，
> 北京地区装修成本最高，平均为 20 万元/90平米；
> 上海次之，为 18 万元/90平米；广州和深圳分别为 16 万元和 15 万元。
```

### 3.3 用户行为洞察

**格式**：
```markdown
> **用户洞察**：基于 [数据来源] 的用户行为分析显示，
> [洞察结论]。[数据支撑]
```

**示例**：
```markdown
> **用户洞察**：基于 2024 年用户调研数据的分析显示，
> 北京地区装修用户在选择装修公司时最关注价格（占比 45%）和服务质量（占比 38%），
> 而装修风格选择偏好中，现代简约风格最受欢迎（占比 52%）。
```

## 四、引用标注要求

### 4.1 必填项

每个引用必须包含：
- **来源名称**：明确的来源机构或个人
- **发布时间**：数据的发布时间（年份或完整日期）
- **来源链接**：可访问的来源链接（如有）
- **数据值**：具体的数据值或结论

### 4.2 可选但推荐项

- **DOI**：学术论文的 DOI（如有）
- **标准编号**：国家或行业标准的编号
- **作者信息**：专家或作者的背景信息

### 4.3 注意事项

1. **链接有效性**：定期检查来源链接是否有效
2. **数据时效性**：标注数据发布时间，过期数据需要更新
3. **准确性**：确保引用的数据值和结论准确无误
4. **格式统一**：使用统一的引用格式，保持一致性

## 五、引用检查清单

针对每个页面，检查：

- [ ] 是否所有统计数据都有引用来源？
- [ ] 引用格式是否符合规范？
- [ ] 来源链接是否有效？
- [ ] 数据发布时间是否标注？
- [ ] 数据是否在有效期内（未过期）？
- [ ] 独特见解是否有数据支撑？

## 六、示例：完整的引用段落

```markdown
## 北京装修价格（2024年最新数据）

根据 [北京市住建委 2024 年第三季度报告](https://zjw.beijing.gov.cn/) 数据显示，
北京地区 90 平米住宅平均装修成本为 **15-25 万元**。

### 影响价格的因素

根据 [《住宅装饰装修工程施工规范》(GB 50327-2001)](https://www.mohurd.gov.cn/) 规定，
装修工程必须按照相关标准进行施工和验收，这直接影响装修成本。

> **数据洞察**：基于 2020-2024 年北京市住建委历史价格数据的分析显示，
> 北京地区装修成本呈现逐年上涨趋势，年均涨幅约 5-8%。
> 其中，2024 年相比 2023 年上涨 6.5%。

### 如何控制装修成本

根据 [李四 资深装修设计师](https://example.com/expert) 的观点，
合理规划装修预算，选择性价比高的装修材料，可以有效控制装修成本。
```
